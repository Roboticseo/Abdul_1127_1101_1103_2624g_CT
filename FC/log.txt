20141128 На основе Abd..g.data сделал файл прогнозов:

И внес коррективы для восстановления забойного давления с сентября месяца 2014 года.
Здесь я использовал хитрые операции с UDQ, которые присваиваются контролирующим параметрам

 Abd2624gbase.data - базовый фонд с контролем по давлению с сентября (прогноз с 1 октября)
 Abd2624gFC.data - загтовка для многовариантыных расчетов на прогнозе для случая оптимисчтичной структуры (первоначальной).
 Неопределенности, которые варьируются на прогнозе:
'mkx'	1	0.225	1.8	FLOAT	/ - множитель на тензор проницаемости в районе бурения скважин
'mppcw'	1	0.292	1.4	FLOAT	/ - размер переходной зоны 
'mmulz'	1	0.9	11	FLOAT	/ - распространение фильтрующей перемычки
Все пределы параметров взяты из опыта адаптации модели: строилась гистограмма распределения модификаторов - и далее делается торнадоплот для подсчета влияния каждого из факторов неопределенности.

 Abd2624gFC_saved_sch.inc - новый шедуль (здесь учтен реку формат, но так как навигатор сейчас не может номрально посчитать данный прогноз - скважина w4 просот не включается... я экспортировал реку в  шедуль).
